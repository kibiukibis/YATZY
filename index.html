  • New game
  • Rules
    • You have three rolls to get your desired combination
    • Dice combinations are in Reminder section
    • If dice fits you hold it by pushing on it
    • Yoy can unhold it too
    • To take score push on desired table place
    • If total score of [aces, ...,sixes] >63, get 50 bonus
    • Game ends when all table lines are filled
    • Your goal is to get more points
  • Reminder
    • From Aces to Sixes count and add only one of these
    • One pair: For example: sum: 12
    • Two pairs: For example: sum: 2x6+2x3=18
    • 3 of a kind: For example: sum: 4x3=12
    • 4 of a kind: For example: sum: 5x4=20
    • Small straight: For example sum:15
    • Large straight: For example sum:20
    • House: For example sum:5x3+6x2=27
    • Chance: Any combination, for example sum:5x2+6+4+3=23
    • Yatzy: Five equal dices. For example You get 50
